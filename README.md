# 🚀 AWS Boto3 Automation Project

## 📌 Project Overview

This project demonstrates AWS infrastructure automation using Python boto3.  
Instead of manually creating resources using the AWS Console, all AWS services are provisioned programmatically using Python scripts.

This project follows basic Infrastructure as Code (IaC) principles and helps understand cloud automation and DevOps workflows.

---

## 🎯 Objective

- Automate AWS resource provisioning using Python
- Learn AWS SDK (boto3)
- Understand IAM, S3, and EC2 services
- Reduce manual cloud setup effort
- Gain practical DevOps and cloud automation experience

---

## ⚙️ Technologies Used

- AWS (IAM, S3, EC2)
- Python
- boto3 (AWS SDK for Python)
- AWS CLI

## 🚀 Implementation Steps

### 1️⃣ IAM User Creation
An IAM user was created using a Python boto3 script to manage AWS services programmatically.

---

### 2️⃣ S3 Bucket Creation
An S3 bucket was created using boto3 for cloud storage operations.

---

### 3️⃣ EC2 Instance Launch
An EC2 instance was launched using boto3 with proper configuration and tagging.

---

### 4️⃣ AWS Console Verification
All created resources were verified successfully in the AWS Management Console.
